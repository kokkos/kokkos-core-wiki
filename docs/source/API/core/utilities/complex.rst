``Kokkos::complex``
===================

.. role:: cppkokkos(code)
    :language: cppkokkos

Header File: ``<Kokkos_Core.hpp>``

Usage
-----

.. code-block:: cpp

    Kokkos::complex<double> a,b;
    a.imag() = 5.0; a.real() = 1.0
    b = a;
    a += b;
